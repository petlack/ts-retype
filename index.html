Discover duplicate types in TypeScript code

TS retype statically analyzes code and searches for multiple declarations of the same Literal Types, Function Types and Enums/Unions.

Run TS retype inside your project folder to get an HTML/JSON report.

npmyarnnpx
$npx ts-retype .
DocsSource

Install

Install as a dev dependency

npmyarn
$npm add -D ts-retype

or install globally

npmyarn
$npm install -g ts-retype

Usage

To create a report for your project, run

bashnpx
$ts-retype .

Then open the report HTML file (this file is self contained and offline)

$open retype-report.html

CLI

Configuration can be done by either CLI options

ts-retype [options] <path-to-project>
  • -c, --config [file-path]
  • load config - if no path provided, loads .retyperc from current directory. if not set, use default config
  • -e, --exclude [glob...]
  • glob patterns that will be ignored
  • -g, --init [file-path]
  • initializes with default config. if no path is provided, creates .retyperc in the current directory
  • -i, --include [glob...]
  • glob patterns that will be included in search
  • -j, --json <file-path>
  • file path to export JSON report to. if not set, does not export JSON.
  • -n, --noHtml
  • if set, does not export HTML
  • -o, --output <file-path|dir-path>
  • HTML report file path - if provided with dir, create index.html file inside the dir

Or by using the --config option and providing path to config .retyperc

$# generate .retyperc in the current directory$ts-retype -i$# run ts-retype using .retyperc in the current directory$ts-retype -c .

An example of a .retyperc file

{  "exclude": ["**/node_modules/**","**/dist/**"],  "include": ["**/*.{ts,tsx}"],  "json": null,  "noHtml": false,  "output": "./retype-report.html",  "rootDir": "."}

ts-retype

You can also run it programatically, using ts-retype package.

import { scan } from '@ts-retype/retype';const { data, meta } = scan({  exclude: ['**/node_modules/**', '**/dist/**'],  include: ['**/*.{ts,tsx}'],  rootDir: '.',});console.log(meta);for (const dup of data) {  console.log(dup.group, dup.names, dup.files);}

The input for scan is of type ScanProps

export type ScanProps = {  exclude: string[];  include: string[];  rootDir: string;};

An example for the snippets in the landing page would look like this

import { TypeDuplicate } from '@ts-retype/retype';export const duplicate: TypeDuplicate = {  names: [    { name: 'User', count: 1 },    { name: 'IUser', count: 1 },    { name: 'anonymous', count: 1 }  ],  files: [    {      name: 'User',      type: 'literal',      src: '...omitted...',      pos: [54, 117],      offset: 26,      lines: [6, 10],      properties: [        { name: 'displayName', type: 'string' },        { name: 'email', type: 'string' },        { name: 'password', type: 'string' }      ],      file: 'src/model.ts'    },    {      name: 'IUser',      type: 'interface',      src: '...omitted...',      pos: [14, 93],      offset: 7,      lines: [3, 7],      properties: [        { name: 'displayName', type: 'string' },        { name: 'email', type: 'string' },        { name: 'password', type: 'string' }      ],      file: 'src/auth.ts'    },    {      name: 'anonymous',      type: 'literal',      src: '...omitted...',      pos: [195, 266],      offset: 33,      lines: [18, 22],      properties: [        { name: 'displayName', type: 'string' },        { name: 'email', type: 'string' },        { name: 'password', type: 'string' }      ],      file: 'src/api.ts'    }  ],  group: 'renamed',  properties: [    { name: 'displayName', type: 'string' },    { name: 'email', type: 'string' },    { name: 'password', type: 'string' }  ]};

The return type of scan is an array of TypeDuplicate

export type TypeDuplicate = {  files: {    name: string;    file: string;    lines: [number, number];    pos: [number, number];    offset: number;    type: 'interface' | 'literal' | 'alias' | 'function' | 'enum' | 'union';    src: string;    srcHgl?: TokenRoot;    properties?: { name: string; type: string }[];  }[];  group: 'different' | 'renamed' | 'identical';  names: { count: number; name: string }[];  members?: string[];  parameters?: { name: string; type: string }[];  properties?: { name: string; type: string }[];  returnType?: string;  signature?: {    name?: string;    params: { name: string; type?: string }[];    return?: string;    strMin?: string;    strFull?: string;  };  types?: string[];};
